# Recommended IDE

## Setting up PyCharm
1. Check if Python is installed  
- Mac/Linux: At the terminal type: python3 --version 
- Windows: At the command prompt type: python --version 
- If a version is displayed then skip step 2.
2. Install Python 
- Download Python from [here](https://www.python.org/downloads/) 
- Run the downloaded file to install Python. 
3. Download Pycharm 
- Download from [here](https://www.jetbrains.com/help/pycharm/installation-guide.html#standalone)
4. Install Pycharm
- Follow the installation guide [here](https://www.jetbrains.com/help/pycharm/installation-guide.html#standalone)
5. Create Folders
- Create a folder named CITS2200 to store all lab files.
- Inside CITS2200, create subfolders for each lab (e.g. Lab_1 for Lab 1).
6. Download the Lab files 
- Download the lab files for each lab into their corresponding folder.
7. Create a Pycharm Project
- Open Pycharm and in the project section click 'New Project'.
- Click the folder icon to the right of 'Location' and then select the 'CITS2200' folder.
- Ensure the Interpreter type is set to Project venv and select a Python version. 
- Click create. A warning will show stating the directory is not empty, click 'Create from Existing Sources'.
- Instructions for creating a project can also be found [here](https://www.jetbrains.com/help/pycharm/creating-and-running-your-first-python-project.html#prerequisites)
